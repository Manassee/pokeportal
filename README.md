# 🧭 PokePortal

**PokePortal** ist ein modernes Web-Praxisprojekt rund um das Pokémon-Universum.  
Ziel des Projekts ist es, **HTML, CSS, JavaScript und React** nachhaltig und praxisnah zu erlernen, indem eine funktionsreiche Webplattform aufgebaut wird – inspiriert vom offiziellen Pokémon-Portal.

Das Projekt dient gleichzeitig als **Lernprojekt**, **Portfolio-Projekt** und als langfristige **Frontend-Referenz**.

---

## 🎯 Projektziele

- Sicherer Umgang mit **HTML5, CSS3 und JavaScript**
- Einstieg und Vertiefung in **React**
- Saubere Projekt- und Komponentenstruktur
- Arbeiten mit echten APIs
- Entwicklung einer responsiven, performanten Web-App
- Vorbereitung auf professionelle Frontend- & Fullstack-Projekte

---

## 🧩 Features

### ✅ Basis
- Pokédex-Übersicht
- Detailansicht einzelner Pokémon
- Suche & Filter (Name, Typ, Generation)
- Responsives Design (Mobile, Tablet, Desktop)

### 🚀 Geplante Erweiterungen
- Favoriten-System
- Pokémon-Vergleich
- Dark / Light Mode
- Animationen & Transitions
- Offline-Modus (PWA)
- Mehrsprachigkeit (DE / EN)
