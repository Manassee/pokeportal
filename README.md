# pokeportal
PokePortal ist ein modernes Web-Projekt rund um das Pokémon-Universum. Ziel des Projekts ist es, HTML, CSS, JavaScript und React praxisnah zu erlernen, indem eine funktionsreiche Pokémon-Plattform entwickelt wird, inspiriert vom offiziellen Pokémon-Portal.  Das Projekt dient gleichzeitig als Lernprojekt.
